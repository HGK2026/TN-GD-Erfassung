# GD-Erfassung

Eine einfache Web-App zur Erfassung von Besucherzahlen und Notizen für Gottesdienste. Die App speichert Daten in einer Firebase Realtime Database und bietet Live-Synchronisation sowie PDF-Export.

Wichtig:
- Die Firebase-Konfiguration ist in `index.html` enthalten. Stelle sicher, dass du deine Realtime Database Sicherheitsregeln anpasst, damit nicht jeder schreiben kann.

Benutzung:
- Einfach `index.html` im Browser öffnen (oder auf einem Webserver bereitstellen).
- Optional: Deploy auf GitHub Pages.
